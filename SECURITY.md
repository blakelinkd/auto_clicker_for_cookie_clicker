# Security Policy

## Reporting a Vulnerability

If you find a security issue, please open a GitHub issue or contact blakelinkd@gmail.com.

## Scope

This bot automates gameplay for the Steam version of Cookie Clicker. It:
- Simulates mouse clicks and keyboard input
- Reads game state via an in-game mod
- Does not modify game files or communicate with external servers

**Use at your own risk.** Automating gameplay may violate Cookie Clicker's terms of service.
